## Personal Blog Platform

This project is a dynamic, single-page blog application developed using HTML5, CSS3, and vanilla JavaScript. It provides a comprehensive interface for content creation, allowing users to publish text-based posts accompanied by image uploads. The application features a dark-themed, responsive design and incorporates full CRUD (Create, Read, Update, Delete) functionality, enabling users to manage their content seamlessly without page reloads.

The application leverages the Web Storage API to maintain data persistence, ensuring that all blog posts and engagement metrics remain available across browser sessions. Key interactive features include a real-time like system, an inline editing module, and the use of the FileReader API for local image processing. By relying entirely on native browser technologies, the project demonstrates an efficient approach to state management and DOM manipulation in a lightweight, framework-free environment.
